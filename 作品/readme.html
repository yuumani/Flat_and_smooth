なめらかな面

今回はなめらかな面を手に取っていただきありがとうございます。

このゲームの実行方法

index.htmlを実行してください。

このゲームの遊び方

この世界には摩擦はありません。そのため、一度動き出すと障害物に当たるまで止まることができません。その中で、左上にいるキャラクターを右下の目的地 （緑色）のブロックを目指してください。
また、時々制限時間が変化しますのでご注意ください。

操作方法

PC・Macの場合（ハードキーボードがある場合）

十字キーを用いて上下左右にキャラを操作してください。

なお、PCでもパッドをクリックまたはタップすることによっての操作もできます。

スマホ・タブレットの場合（ハードキーボードがない場合）

左下にある十字のパッドをタップして操作してください。

推奨環境

動作確認済みのブラウザ

HTML5およびJavaScriptが実行可能なブラウザで動作します。ただし、動作保証はしません。Android,iOSではUSBデバッグを除くオフライン起動には対応していません。今後GitHubにアップロードし、これらのモバイルデバイス向けOSにも対応予定です。
